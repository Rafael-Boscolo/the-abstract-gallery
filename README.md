## Projeto
Este é um projeto web desenvolvido para mostrar uma estilização em imagem em forma de grid.

## Tecnologias
* HTML
* CSS
* JS

## Sobre
Projeto desenvolvido é uma galeria de fotos web, com geração de foto aleátoria sem perca de resolução
